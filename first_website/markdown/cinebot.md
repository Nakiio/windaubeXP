# 🎬 CineBot

Un chatbot intelligent spécialisé dans la recommandation de films et séries, propulsé par une IA capable de comprendre vos envies du moment.

## 🚀 Aperçu du Projet
Développé dans le cadre du projet ELIZA, CineBot utilise la technologie RAG (Retrieval-Augmented Generation) pour analyser un dataset de films et générer des recommandations humaines et pertinentes.

### 🕹️ Fonctionnalités
* **Recherche Contextuelle :** Recommandations basées sur l'humeur ou des préférences spécifiques (ex: "un film comme Snowfall").
* **Système RAG :** Analyse précise d'un catalogue incluant Netflix, Disney, Hollywood et des Animes.
* **Interface Multi-canal :** Utilisable via une interface Web moderne ou directement sur Discord.

## 🛠️ Stack Technique
* **IA / RAG :** LlamaIndex & LLaMA 3.2 (Ollama)
* **Backend :** Python & Flask
* **Frontend :** HTML, CSS, JavaScript
* **Déploiement :** Docker & Docker Compose
