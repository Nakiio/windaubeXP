# 🎵 Melovibe

Une application web de découverte musicale inspirée des réseaux sociaux à défilement vertical, dédiée à l'immersion sonore.

## 🚀 Aperçu du Projet
Melovibe propose un flux continu de musique permettant aux utilisateurs d'écouter, de liker et de commenter des titres en temps réel. L'objectif est de rendre la découverte musicale fluide et interactive.

### 🕹️ Fonctionnalités
* **Infinite Scrolling :** Un thread musical qui défile à l'infini pour une expérience sans coupure.
* **Intégration API Deezer :** Accès à un catalogue immense avec extraits audio et visuels haute qualité.
* **Interactions Sociales :** Système complet de likes et de commentaires sur les morceaux.
* **Gestion de Profil :** Inscription et authentification sécurisée (JWT) pour sauvegarder ses préférences.

## 🛠️ Stack Technique
* **Frontend :** HTML5, CSS, JavaScript
* **Backend :** Node.js (Express)
* **Base de données :** MySQL
* **API :** Deezer API
