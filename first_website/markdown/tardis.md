# 🚄 SNCF TARDIS Project

Un tableau de bord interactif d'analyse et de prédiction des retards de trains sur le réseau SNCF.

## 🚀 Aperçu du Projet
Le projet TARDIS (Train Delay Analysis & Prediction) analyse les causes des retards sur de nombreuses routes françaises et internationales. Il permet de comparer les performances des lignes et de visualiser les données de circulation.

### 🕹️ Fonctionnalités
* **Analyse de Retard :** Étude des motifs de retard (départ vs arrivée) et de leurs causes.
* **Dashboard Interactif :** Sélection de gares pour visualiser les statistiques mensuelles.
* **Cartographie :** Affichage des coordonnées géographiques des gares sur une carte interactive.
* **Modèle Prédictif :** Utilisation du machine learning pour anticiper les comportements de retard.

## 🛠️ Stack Technique
* **Interface :** Streamlit
* **Analyse de données :** Python, Pandas
* **Machine Learning :** Scikit-learn & Pickle pour la sauvegarde du modèle
